How to run me?

1. Pull the repo, download as zip file.
2. Extract zip file to C:\. You should see all files under, C:\soFiChallenge-master. Then change it to, C:\soFiChallenge.
3. Open test_class.py in pytho IDE, I used pycharm.
4. Execute it.
5. It should run all the test_case, both positive and negative.
6. print out result in the IDE console.
