How to run me?

1. Pull the repo, download as zip file.
2. Extract zip file to C:\. You should see all files under, C:\soFiChallenge-master.
3. Execute in IDE: open test_class.py in pytho IDE, I used pycharm.
4. Execute in command prompt, cmd. 
5. It should run all the test_case, both positive and negative.
6. print out result in the IDE console or windows cmd
